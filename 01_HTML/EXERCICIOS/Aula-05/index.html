Aula 05

Com relação ao trecho de código abaixo responda: 

                                                            
      <!DOCTYPE html>
    <html lang="pt-br">
    <head>
        <meta charset="UTF-8">
        <title>Estrutura</title>
    </head>
    <body>
        Olá mundo!
    </body>
    </html>

a) Para que serve a tag <!DOCTYPE html>?

A tag <!DOCTYPE html> define o tipo de documento, é uma declaração para instruir o navegador sobre a versão do HTML utilizada no arquivo

b) Para que serve as meta tags <meta>?

A tag <meta> define metadados sobre um documento HTML (Metadados são informações sobre dados). Meta tags são linhas de código HTML que descrevem o conteúdo do seu site facilitando o rastreamento dos mecanismos de pesquisas por sites relevantes quando uma busca é realizada.

c) Qual a importância de usar a configuração de charset?

O Charset é o conjunto de caracteres utilizados para escrever o documento.

d) Com o conhecimento adquirido até o momento, o que poderia ser melhorando nesse trecho de código?

No meu ponto de vista, pode ser adicionado duas tags "<meta>" antes do <title>, e também envelopar a frase "Olá mundo!" dentro de uma tag <p>, para não ficar solta.
